Bid Calculator

Instructions:
- Download the .zip file below
- Unzip it. It will give you a folder called BidCalculator and put it wherever you find convenient.
- Inside this folder, you will see a file called main.exe. Clicking on this file will start the application. There is also a folder called _internal. Don't mess with the files inside of it.
- The first time the application is run, it might take a second. And your computer might think it's a virus (I promise it isn't).
- Anytime you want to run the application, go into this folder and click main.exe
- main.exe must always be in the folder with _internal. In other words, don't move main.exe into another spot without _internal with it.
- If the app breaks or isn't behaving as expected, let me know.
